## Installing and using mitiq 

*[mitiq](https://mitiq.readthedocs.io/en/stable/)* is licensed under the ***[GNU GPLv3](https://github.com/unitaryfoundation/mitiq/blob/main/LICENSE)*** license, and *mitiq* is <u>**not**</u> distributed or installed by default with *amazon-braket-examples*. For developers of this repo, or related, do not include *mitiq* as a requirement in the main requirements.txt file. 

Please note, the `on_mitiq` folder is completely standalone. 

>[!WARNING]
>The files in the directory are **examples** of how to use *mitiq* with Amazon Braket. Please do not naively copy files for use in production workflows.
